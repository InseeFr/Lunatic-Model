# Projets qui utilisent Lunatic-Model

## Eno

[Eno](https://github.com/InseeFr/Eno) utilise les objets Lunatic-Model pour générer les questionnaires Lunatic
(au format json).

## Données au format Lunatic

Voir [la documentation sur les données Lunatic](../data/README.md).

⚠️ Les conversion de données entre xml et json ne sont plus supportées depuis la version `3.0.0` de Lunatic-Model.

Certaines briques applicatives utilisent encore les conversion de données à partir de versions `2.y.z`.
