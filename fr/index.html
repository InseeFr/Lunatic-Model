Documentation de Lunatic-Model

Introduction

Lunatic-Model est une librairie java permettant de transformer un questionnaire au format XML respectant un modèle défini dans ce projet, dans un autre format tel que le format JSON. Lunatic-Model s'interface avec l'application Eno et est destiné à alimenter les composants définis dans la bibliothèque Lunatic
